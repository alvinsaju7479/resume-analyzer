# 🤖 AI Resume Analyzer

## 🚀 Overview

An AI-powered web application that analyzes resumes and compares them with job descriptions to calculate an ATS (Applicant Tracking System) score and identify skill gaps.

---

## 🧠 Features

* 📄 Resume parsing (PDF)
* 🧠 Skill extraction using NLP (regex-based matching)
* 📊 ATS score calculation
* 📈 Skill comparison visualization
* 📂 Category-wise skill analysis
* 💡 Smart suggestions for improvement

---

## 🛠 Tech Stack

* Python
* Streamlit
* spaCy
* Scikit-learn
* Matplotlib
